# fontsize
jquery.fontsize.js plugin to control fontsize in target dom

# START

- 引入jquery

`<script src="lib/jquery-1.12.min.js"></script>`

- 引入插件

`<script src="dist/fontsize.min.js"></script>`

# 初始化

```
$('#view').FontSize({
    increaseBtn: '#big',
    reduceBtn: '#small'
});
```
